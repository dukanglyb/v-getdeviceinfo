# Vue JavaScript 获取设备信息

 ### [在线地址](https://dukanglyb.github.io/v-getdeviceinfo/)

### JS获取设备信息

 - DeviceType: 设备类型
 - OS: 操作系统
 - OSVersion: 操作系统版本
 - ScreenHeight: 屏幕高
 - ScreenWidth: 屏幕宽
 - Language: 当前使用的语言-国家
 - NetWork: 联网类型
 - Orientation: 横竖屏
 - BrowserInfo: 浏览器信息
 - Fingerprint: 浏览器指纹
 - UserAgent: 包含 appCodeName,appName,appVersion,language,platform 等
 - GroPosition: 地理位置
 - Date: 系统时间
 - UUID: 通用唯一标识 Universally Unique Identifier
